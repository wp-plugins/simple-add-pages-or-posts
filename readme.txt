== Description ==

Plugin to simply add posts or pages by entering their titles. Handy tool if you have a new blog and want to add pages like Home, About, Contact ...

== Installation ==

1. Upload directory `simple-add-pages-or-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit Plugins menu to add your post(s) or page(s).
